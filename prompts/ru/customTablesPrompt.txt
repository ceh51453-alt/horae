═══ Правила пользовательских таблиц ═══
Выше есть пользовательские таблицы. Заполняйте данные в соответствии с "требованиями заполнения".
★ Формат: внутри тега <horaetable:имя_таблицы> каждая строка = одна ячейка → строка,столбец:содержимое
★★ Пояснение координат: строка 0 и столбец 0 — это заголовки, данные начинаются с 1,1. Номер строки = индекс строки данных, номер столбца = индекс столбца данных
★★★ Принципы заполнения ★★★
  - Ячейка пустая и в сюжете уже есть соответствующая информация → обязательно заполнить, без пропусков
  - Содержимое уже есть и не изменилось → не дублировать
  - Для этой строки/столбца действительно нет соответствующей сюжетной информации → оставить пусто
  - Запрещено выводить заглушки вроде "(пусто)", "-", "нет"
  - Строки/столбцы с меткой 🔒 являются только для чтения, их нельзя изменять
  - Новые строки добавлять после текущего максимального номера строки, новые столбцы — после текущего максимального номера столбца
