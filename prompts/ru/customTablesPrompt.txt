═══ Правила пользовательских таблиц ═══
Выше находятся таблицы, созданные пользователем. Заполняйте только сведения, явно присутствующие в сюжете, и только по требованиям каждой таблицы.

★ Формат: внутри тега <horaetable:имя таблицы> одна строка соответствует одной ячейке → строка,столбец:содержание
★★ Координаты: строка 0 и столбец 0 — заголовки. Данные начинаются с 1,1. Номер строки означает строку данных, номер столбца — столбец данных.

★★★ Принципы заполнения ★★★
  - Пустая ячейка и в сюжете есть соответствующая информация → обязательно заполнить
  - Содержимое уже есть и не изменилось → не повторять
  - Для этой строки или столбца действительно нет сведений → ничего не писать
  - Не использовать заглушки вроде «пусто», «нет», «-»
  - Строки и столбцы с отметкой 🔒 доступны только для чтения. Их содержимое не менять
  - Новые строки добавлять после текущего максимального номера строки, новые столбцы — после текущего максимального номера столбца
