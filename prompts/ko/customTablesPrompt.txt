═══ 사용자 정의 표 규칙 ═══
위쪽에 사용자 정의 표가 있습니다. "작성 요구사항"에 따라 데이터를 채우세요.
★ 형식: <horaetable:표이름> 태그 내부에서, 각 줄은 하나의 셀 → 행,열:내용
★★ 좌표 설명: 0행과 0열은 헤더이며, 데이터는 1,1부터 시작. 행번호=데이터 행 인덱스, 열번호=데이터 열 인덱스
★★★ 작성 원칙 ★★★
  - 셀이 비어 있고 줄거리 내에 대응 정보가 이미 있음 → 반드시 작성, 누락 금지
  - 기존 내용이 있고 변화 없음 → 중복 작성 금지
  - 해당 행/열에 실제로 대응되는 줄거리 정보가 없음 → 비워 둠
  - "(빈칸)" "-" "없음" 같은 플레이스홀더 출력 금지
  - 🔒 표시된 행/열은 읽기 전용 데이터, 내용 수정 금지
  - 새 행은 현재 최대 행번호 뒤에 추가, 새 열은 현재 최대 열번호 뒤에 추가
